Health Care

# 概要

一人一人個人で自分の健康を管理していく上で、管理しやすいようなアプリケーションを目指しました。

# 制作背景（意図）

自分自身の健康を管理する目的で作成しました。
情報は至る所に溢れているため、知らなければ調べれば良いのですが、情報が一つにまとまっていることで管理しやすくなると考えました。
また、健康状態に不安がある場合など、入力した個人のデータを元に病院や産業医からの生活指導などにも活用できると考えました。

# テーブル設計

## users テーブル

| Column            | Type   | Options                  |
| ----------------- | ------ |------------------------- |
| nickname          | string | null: false              |
| email             | string | null: false, unique: true|
| encrypted_password| string | null: false              |
| profile           | text   |                          |
| birthday          | date   | null: false              |

### Association

- has_many :weights
- has_one :weight_goal


## weights テーブル

| Column          | Type       | Options                        |
| --------------- | ---------- | ------------------------------ |
| day             | date       | null: false                    |
| weight          | integer    | null: false                    |
| user            | references | null: false, foreign_key: true |

### Association

- belongs_to :user
- belongs_to :weight_goal


## weight_goals テーブル

| user               | reference| null: false, foreign_key: true |

### Association
- belongs_to :user
